# Estruturas de dados em C#

As seguintes estruturas de dados foram implementadas:
  1. Linked List: Lista duplamente encadeada com suporte a remoção por índice.
  2. Stack e Queue: Implementações dinâmicas utilizando nós.
  3. Hash Table: Tratamento de colisões por chaining (funciona como uma pilha dinâmica, a colisão mais recente fica na frente).
  4. Tree: Árvore genérica com algoritmos de busca DFS (Profundidade) e BFS (Largura).
